About This Project

The Student Performance Analytics Portal is a frontend-based web application designed to help visualize and manage student academic performance in an easy and interactive way.

This project includes multiple pages such as Home, Dashboard, Reports, and Contact form. It is built using HTML5, CSS3, and basic JavaScript to demonstrate layout design, responsiveness, and user interaction.

The main purpose of this project is to understand how real-world dashboards are structured, how data is displayed in tables and cards, and how user input forms work with validation.

In future versions, this project can be extended using backend technologies like Node.js, Express, and MongoDB to make it fully dynamic.
